# xonnel-unpack

A python library: xonnel-unpack

- unpack (.zip   →    dir)
- unpack (.zip   →   file)
- unpack (.7z    →    dir)
- unpack (.7z    →   file)
- unpack (.rar   →    dir)
- unpack (.rar   →   file)
- unpack (.tar   →    dir)
- unpack (.tar   →   file)

## Install

LOCAL:
cd xonnel-unpack
pip install -e .

PyPI:
pip install xonnel-unpack
