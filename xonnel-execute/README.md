# xonnel-execute

A python library: xonnel-execute

- executing files of any type
    - .exe (execute as program)
    - .py  (execute as python script)
    - .cmd (execute as command)
    - .ps1 (execute as powershell script)
    - .bat (execute as batch script)
    - ...  (execute if possible or raise error if not executable)

## Install

LOCAL:
cd xonnel-execute
pip install -e .

PyPI:
pip install xonnel-execute
