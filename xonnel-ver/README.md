# xonnel-ver

A python library: xonnel-ver

- version standarizing
    - standarises all kinds of input to:
        - major
        - minor
        - patch
        - build

    up:
        - major
        - minor
        - patch
        - build

    down:
        - major
        - minor
        - patch
        - build

    compare al of:
        - <
        - >
        - ==
        - <=
        - >=

## Install

LOCAL:
cd xonnel-ver
pip install -e .

PyPI:
pip install xonnel-ver
